1. 委托允许捕获对方法的引用并传递。
2. 通过delegate声明一个委托，不需要实例化。
3. 直接可将函数作为参数传入一个函数。
3. Lambda表达式分为语句Lambda和表达式Lambda
4. 语句Lambda由形参列表 + Lambda操作符=> + 代码块构成，由于委托的类型已确定，形参的数据类型可省略。
~~~
    (first,second) => {
        int i=0;
        return first<second;
        }
~~~
5. 表达式Lambda：只有一个return语句，不可有代码块。
~~~
    (first,second)=> first < second
~~~
6. Lambda表达式没有类型，故不可以把整体当中一个返回值使用。
